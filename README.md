# 不審者・治安マップ

警察庁が公開する「犯罪統計資料」（e-Stat）をもとに、都道府県ごとに不審者関連の犯罪種別（殺人・強盗・放火・不同意性交等・略取誘拐・人身売買・不同意わいせつ・ひったくり）がどの程度発生しているかを地図上で可視化するWebアプリです。

## 使い方

ビルド不要の静的サイトです。リポジトリ直下でHTTPサーバーを立てて `web/index.html` を開いてください（`fetch` で `data/*.json` を読み込むため `file://` では動作しません）。

```sh
python3 -m http.server 8000
# ブラウザで http://localhost:8000/web/index.html を開く
```

## 画面の見方

- **地図**: 都道府県を選択カテゴリの値で色分け（濃いほど件数が多い）。クリックで詳細パネルに犯罪種別ごとの内訳を表示。
- **表示単位の切り替え**: 「認知件数」（実数）と「人口10万人あたり」（人口比、都道府県間の公平な比較用）を切り替え可能。
- **全国ランキング**: 選択中のカテゴリ・単位での上位10都道府県。

## データソース

- 犯罪統計: 警察庁「犯罪統計資料」(暫定値) 令和8年1〜5月分
  https://www.e-stat.go.jp/stat-search/files?page=1&layout=datalist&lid=000001483773
  第6表（重要犯罪・重要窃盗犯 都道府県別 認知・検挙件数・検挙人員 対前年比較）から、不審者による被害に関連性が高い犯罪種別を抽出。
- 人口: 総務省統計局「令和2年国勢調査」（都道府県別人口、正規化の基準値として使用）
- 地図データ: [dataofjapan/land](https://github.com/dataofjapan/land) の `japan.geojson` を `mapshaper` で簡略化（容量13MB→63KB）。簡略化処理でポリゴンの周回方向（winding order）が壊れていたため、d3-geoの仕様（外周は反時計回りではなく、符号付き面積が負になる向き）に合わせて手動で補正済み（`data/japan.geojson`）。

警視庁「メールけいしちょう オープンデータ」は東京都内のみが対象で、不審者の声かけ等を含む詳細な事案データを公開していますが、全国47都道府県を横断的に比較できる統一フォーマットではないため、今回は採用していません。都道府県をまたいだ比較を行うため、警察庁が全国統一フォーマットで公開している犯罪統計資料を採用しています。

## データの再生成

`data/raw/crime_stats_r8_1-5.csv` は上記e-Statページからダウンロードした生データ（Shift_JIS）です。最新のデータに更新する場合は、e-Statから最新の犯罪統計資料CSVを取得して `data/raw/` に配置し、`scripts/build_data.py` 内の `RAW_CSV` パスやファイル名、必要に応じて `data/meta` のラベルを更新したうえで以下を実行してください。

```sh
python3 scripts/build_data.py
```

`data/prefectures.json` が再生成されます。

## ディレクトリ構成

```
data/
  raw/crime_stats_r8_1-5.csv   生データ（e-Stat, Shift_JIS）
  prefectures.json              都道府県×カテゴリの集計済みデータ（自動生成）
  japan.geojson                 都道府県境界の簡略化済み地図データ
scripts/
  build_data.py                  生データから prefectures.json を生成するスクリプト
web/
  index.html / style.css / app.js  フロントエンド（D3.js, ビルド不要）
  vendor/d3.v7.min.js             D3.js本体（CDN依存を避けるためローカルに同梱）
```

## 注意事項

- 犯罪統計資料の数値は暫定値です（確定値は翌年2月に公開）。
- 件数の単純比較は人口の多い都道府県ほど大きくなりやすいため、「人口10万人あたり」表示も併用してください。
- 「不審者」を直接分類した全国統一の統計区分は存在しないため、不審者による被害に関連性が高いと考えられる犯罪種別（重要犯罪等）を代理指標として使用しています。
